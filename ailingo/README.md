# AiLingo (アイリンゴ)

Claudeを効率的に使いこなす技術をゲーム感覚で習得するアプリ。

## セットアップ

```bash
npm install
cp .env.example .env.local
# .env.local に Firebase 設定を記入
npm run dev  # http://localhost:5174
```

## 学習ステージ

- **DOJO** 🏯 — ワンショットプロンプトの7つの型を7日間で習得
- **BUILDER** 🏗️ — Easy〜Masterの難易度別アプリを5時間セッション内で完成
- **CREATOR** 🌟 — 自由設計で本格アプリを創る

## Tech Stack

React 18 · TypeScript · Vite · Tailwind CSS · Firebase · Claude API (Haiku)
